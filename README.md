# Beyblade X Database

A full-stack Beyblade X catalog and collection tracker built with Laravel, React, PostgreSQL, and Supabase.

## Overview

This project helps Beyblade X players:

- manage catalog items such as blades, ratchets, and bits
- create official setups
- save combinations they own
- track their own Beyblade collection

## Features

- Blade, ratchet, and bit catalog CRUD
- Official setup creation
- React form for creating official setups
- PostgreSQL database via Supabase
- Docker-based local and deployment setup

## Tech Stack

- Laravel
- React
- Vite
- Tailwind CSS
- PostgreSQL
- Supabase Storage
- Docker
- Render deployment target
